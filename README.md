# 🌦️ Weather Dataset Analysis Project

## 📌 Project Overview
This project focuses on analyzing an Indian climate dataset (2024–2025) using Python. The dataset contains important weather parameters such as temperature, humidity, and rainfall across different cities and states.

The goal of this project is to extract, process, and understand climate data using basic data analysis techniques.

## 📂 Dataset Details
The dataset includes the following columns:

- Date  
- City  
- State  
- Temperature_Max (°C)  
- Temperature_Min (°C)  
- Temperature_Avg (°C)  
- Humidity (%)  
- Rainfall (mm)  

---

## ⚙️ Project Workflow

1. Import required libraries (Pandas, NumPy)
2. Load the dataset using Pandas
3. Extract individual columns
4. Convert data into NumPy arrays
5. Display and analyze values

---

## 💡 Key Features

- Easy dataset loading using Pandas  
- Conversion of data into NumPy arrays  
- Separate handling of each weather parameter  
- Clean and simple data processing approach
